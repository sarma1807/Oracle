### WARNING : THIS METHOD WILL REQUIRE APPLICATION DOWN TIME

```
Migrate
FROM : Oracle 11.2.0.4    Grid Infrastructure and RDBMS
  TO : Oracle 19.11.0.0.0 Grid Infrastructure and RDBMS
```
